A Java-based Bank Management System desktop application developed using Java Swing and MySQL. This project simulates real-world banking operations through a simple and user-friendly graphical interface.

**Features**

User Signup and Login using Card Number & PIN

Deposit and Withdraw money

Fast Cash transactions

Balance Enquiry

Mini Statement generation

Secure handling of banking transactions

Technologies Used

Java

Java Swing (AWT)

MySQL

JDBC

NetBeans IDE

Database

Database Name: bankmanagementsystem

Tables Used:

login

signup

bank

How to Run

Open the project in NetBeans

Create the MySQL database and required tables

Update database credentials in Conn.java

Run Login.java
